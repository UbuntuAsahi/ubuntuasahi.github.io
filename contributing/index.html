Contributing to Ubuntu Asahi

Thank you for taking the time to contribute! Bug reports and pull requests are very welcome.

Ubuntu Asahi is in early development and our contribution process is loose. Following Asahi Linux’ style is encouraged. Ubuntu Asahi discussion primarily takes place on the #ubuntu-asahi channel of OFTC.

This project and everyone participating in it are governed by the Ubuntu Asahi Code of Conduct. Please report unacceptable behavior to conduct@ubuntuasahi.org.
